Politique de Confidentialité

Date de prise d'effet: May 08, 2019

Cette Application facebook n'a pas pour objectif d'être diffusée et d'être publique. Cette application est utilisable à des fins personnelles dans le cadre de travaux universitaires. Cette application Facebook ne sera utilisable que par son propriétaire. À cet effet, les données des utilisateurs ne sont pas collectées.

Cependant, le code du logiciel est disponible en open-source. Le logiciel n'est fonctionnel que si vous créez votre propre application facebook en fournissant votre propre app token, avec votre propre politique de confidentialité.

Définitions

  • Logiciel

    Par Logiciel j'entends le programme javascript permettant de parcourir une page facebook publique, qui est dépendant de cette Application facebook

  • Application facebook

    Par Application facebook j'entends l'application facebook qui est lié à cette présente Politique de Confidentialité

  • Utilisateur

    Seul le propriétaire et créateur de cette application peut être l'utilisateur.

Collecte et utilisation des données et Types de données recueillies

Les seules données traitées sont des données publiques, provenant de pages facebook publiques. En aucun cas les données de l'Utilisateur sont récupérées et analysées. Seules les données retournées par la Graph API de facebook d'une quelconque page facebook seront extraites. Ces données traités ne sont utilisés que dans le cadre d'un travail universitaire
